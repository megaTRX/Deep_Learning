🧠 Deep Learning Curriculum Journey
15주 만에 퍼셉트론부터 생성형 AI까지: 딥러닝 마스터 클래스

이 저장소는 딥러닝의 기초부터 최신 생성 모델 및 강화학습까지, 15주간의 치열한 학습 여정을 기록하는 공간입니다. 단순히 코드를 복사하는 것이 아니라, 구조를 이해하고 구현하는 것에 집중합니다.

주차,주제,핵심 내용,비고
01,OT,"강의 소개 및 개발 환경 설정 (Python, Colab)",🚀 Start
02,퍼셉트론 (Perceptron),"신경망의 기초, 논리 게이트 구현",
03,MLP (다층 퍼셉트론),역전파(Backpropagation)와 활성화 함수,
04,MLP & Keras,Keras 라이브러리를 활용한 모델링 실습,
05,심층 신경망 (DNN),"과적합 방지, 가중치 초기화, 배치 정규화",
06,컨벌루션 신경망 (CNN),"합성곱 계층, 풀링 계층의 원리",
07,영상인식,"ResNet, VGG 등 주요 CNN 아키텍처 분석",
08,순환신경망 (RNN),시퀀스 데이터 처리의 기초,Mid-term
09,RNN 실습,"LSTM, GRU를 이용한 시계열 예측",
10,자연어 처리 (NLP),"Word Embedding, Attention Mechanism",
11,GPT & 생성형 모델,Transformer 구조와 GPT 모델의 이해,
12,생성 모델 (Generative),"AutoEncoder, VAE의 원리",
13,이미지 생성 모델,"GAN(Generative Adversarial Networks), Diffusion",
14,강화학습 (RL),"Agent, Environment, Reward의 상호작용",
15,기말고사,최종 프로젝트 및 이론 정리,Final

🛠️ 기술 스택 (Tech Stack)
Language: Python 3.2
Library: TensorFlow, Keras, PyTorch
Environment: Google Colab, Jupyter Notebook
Concepts: Computer Vision, NLP, Reinforcement Learning
